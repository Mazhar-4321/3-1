In Html , List is used to display inoformation on the page. There are three types of lists in Html. 1)Ordered lists : In Ol, we can display ordered information.In Ordered list , we have two tags ol and li . ol is the parent tag. In li tag we specify data. Ex: student names,rank wise information 2)Unordered list : In Unorderd list , we display unordered information .In Unordered List, we have two tags ul and li. ul is the parent tag. In li tag we specify data. Ex: books names,city names 3) Description List : In Description List , we have to display the information that contains both description and data. In Description List , we have three tags dl,dt,dd. dl is the parent tag. dt stands for description tag and dd stands for description data. Ex : class & its students
